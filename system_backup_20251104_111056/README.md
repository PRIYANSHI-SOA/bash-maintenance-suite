## Objective
Write a suite of Bash scripts to automate system maintenance tasks such as backup, system updates, and log monitoring.

## Day-wise Tasks
- **Day 1:** Write a script for automated system backups.
- **Day 2:** Create a script to perform system updates and cleanup.
- **Day 3:** Develop a log monitoring script to alert on certain conditions.
- **Day 4:** Combine scripts into a maintenance suite with a menu to execute them.
- **Day 5:** Test scripts and add error handling and logging functionalities.

---
**Usage**
1. Make scripts executable using: `chmod +x *.sh`
2. Run the main menu using: `./maintenance_menu.sh`
