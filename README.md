 Overview

This project is a comprehensive Bash scripting suite  designed to automate essential system maintenance tasks in a Linux environment.  
It simplifies routine administrative activities like backups, updates, cleanup, and log monitoring — improving system efficiency and reducing manual workload.


 Objective

To design and implement a suite of Bash scripts that:
- Automate regular system maintenance tasks.
- Perform backup and cleanup operations.
- Monitor system logs and alert administrators.
- Provide an interactive menu for user-friendly access.

 Features

- Automated Backups – Securely stores critical files.
-  System Cleanup – Removes temporary and unnecessary files.
- Log Monitoring – Detects issues by scanning system logs.
- Menu-Driven Interface – Simplifies script execution for users.
- Error Handling & Logging – Tracks activity and ensures reliability.
